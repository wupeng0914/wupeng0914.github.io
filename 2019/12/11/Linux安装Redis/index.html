发表于 2019-12-11 | 分类于 NoSQL

###Linux环境安装Redis数据库

#####第一步：下载Redis安装包

或直接通过Redis官网进行安装包下载。

第二步：安装
1、解压安装包

2、进入解压后的文件目录并使用”make && make install”命令对解压的Redis文件进行编译并安装

PREFIX=/usr/local/redis用于指定Redis的安装目录（此处指定安装目录为”/usr/local/redis”，需要手动创建该目录）

看到此处证明安装成功。

#####第三步：部署

1、为便于以后对Redis的管理配置，这里我们将Redis中的conf配置和常用bin命令移动到统一目录下

2、将 redis.conf 文件复制到 /usr/local/redis/etc 目录下

######3、将原安装目录下 src 目录中的 mkreleasehdr.sh, redis-benchmark, redis-check-aof ,redis-check-rdb ,redis-cli ,redis-server 等文件复制到 /usr/local/redis/bin 目录下(原本已存在的可不必覆盖)

测试redis启动：

此时，redis已经可以正常启动了。

修改 redis.conf 中的”daemonize”参数为”yes”，使redis可以以后台方式启动：

以指定的配置启动redis：

此时，Redis便已在后台启动了。

现在Redis只能在本服务器进行访问，如果需要允许在外部访问该Redis，可以将redis.conf中的 “bind” 参数值修改为 “0.0.0.0” 即可。

至此，Redis的基本安装过程已经完毕了！
